Chapter 05

데이터관리 기본 명령어

1. describe & codebook

describe

데이터 세트의 형식상 속성을 요약합니다. (변수 개수, 저장 형태, 레이블 등)

sysuse auto, clear
describe price mpg

codebook

데이터의 내용상 속성을 보여줍니다. (범위, 빈도, 결측치, 기술통계 등)

2. display

결과를 보여주는 명령어로, 계산기처럼 유용하게 사용할 수 있습니다.

  • 간단한 계산:

    di 4*(2+3)

  • 상수 값 확인 (원주율):

    di _pi

  • 포맷 지정:

    di %12.10f _pi

  • 확률 계산 (동전 6번 던져 앞면이 2번 나올 확률):

    di comb(6,2)*0.5^2*0.5^4

3. list

데이터세트의 값을 Result창에 그대로 보여줍니다.

Options

  • list, sep(10)
    : 매 10번째 행에 구분선
  • list, mean sum N
    : 평균, 합계, 관측치 수 표시

Qualifier: in

  • list price in 1/10
    : 1~10번째 관측치
  • list price in -5/L
    : 끝에서 5번째부터 마지막 관측치

4. rename

주어진 변수 이름을 변경합니다.

STATA 변수 이름 규칙:

  • 영문자, 숫자, '_' 조합으로만 구성
  • 최대 32글자
  • 한글 사용 불가
  • 숫자로 시작 불가
  • -, * 등 일부 특수문자 사용 불가
  • 공백(space) 사용 불가
sysuse auto, clear
rename price price1000

5. sort, gsort, bysort

자료를 정렬하는 명령어입니다.

sort

오름차순 정렬만 가능. sort price

gsort

내림차순(-), 오름차순(+) 정렬 가능. gsort -price

bysort

by와 sort의 결합. 그룹별로 정렬 후 명령어를 실행.
bysort company : summarize invest

6. drop, keep, move, order

변수나 관찰치를 관리합니다.

drop

변수/관찰치 삭제
drop price mpg
drop in 1/10

keep

변수/관찰치 남기기
keep make price
keep if foreign==1

move

변수 위치 이동 (2개)
move price rep78

order

여러 변수 위치 변경
order mpg rep78 price

7. 데이터 레이블

데이터세트나 변수에 설명을 붙이는 기능입니다.

데이터세트 레이블

label data "Stata연습용파일_5월_2일"

변수 레이블

label variable price "자동차 가격"

👉 describe 명령어로 레이블을 확인할 수 있습니다.

8. generate & replace

generate (gen)

새로운 변수를 생성합니다.

gen gap = flife - mlife
gen ln_pop = log(pop)
gen var2 = _n

replace

기존 변수의 값을 대체합니다.

replace pop = pop*1000
replace var5=2 if unemp>=9.0
