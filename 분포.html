제4장 여러 가지 분포

기초통계학

목차

  • 1. 이산확률분포 (베르누이, 초기하, 이항 분포)
  • 2. 정규분포
  • 3. 표본분포 (표본평균, 중심극한정리, t, $\chi^2$, F 분포)
  • 4. 정규분포 분위수대조도 (Q-Q Plot)

1. 이산확률분포

1.1 베르누이 분포 (Bernoulli Distribution)

어떤 시행의 결과가 '성공(success)'과 '실패(fail)' 두 가지 뿐인 시행을 베르누이 시행이라고 합니다.

  • 성공 확률을 $p$, 실패 확률을 $q = 1-p$로 표기합니다.
  • 베르누이 확률변수는 성공을 1, 실패를 0으로 대응시킨 확률변수입니다.

확률질량함수 (PMF)

$p(x) = p^x(1-p)^{1-x}, \quad x \in \{0, 1\}$

기댓값 및 분산

  • 기댓값: $E(X) = p$
  • 분산: $Var(X) = p(1-p)$

예제 4.1: 동전 던지기

동전의 앞면이 나올 경우를 성공(1)으로 할 때, 성공확률은 $p=1/2$ 입니다.

  • 평균: $E(X) = p = 1/2$
  • 분산: $Var(X) = p(1-p) = \frac{1}{2}(1-\frac{1}{2}) = \frac{1}{4}$

1.2 초기하분포 (Hypergeometric Distribution)

N개의 원소 중 D개가 특정 속성을 가질 때, n개의 표본을 비복원추출하는 경우, 표본에 포함된 특정 속성 원소의 개수(X)에 대한 분포입니다. 각 시행이 독립이 아니라는 특징이 있습니다.

확률질량함수 (PMF)

$p(x) = P(X=x) = \frac{\binom{D}{x}\binom{N-D}{n-x}}{\binom{N}{n}}$

기댓값 및 분산

  • 기댓값: $E(X) = np \quad (p=D/N)$
  • 분산: $Var(X) = np(1-p) \left( \frac{N-n}{N-1} \right)$

예제 4.2: 불량 전구

50개의 전구 중 3개가 불량품일 때, 5개를 비복원추출하여 표본에 포함된 불량품 개수 X의 분포는 초기하분포를 따릅니다.

이때 $N=50, D=3, n=5$ 이며 확률밀도함수는 다음과 같습니다.

$p(x) = P(X=x) = \frac{\binom{3}{x}\binom{47}{5-x}}{\binom{50}{5}}, \quad x=0, 1, 2, 3$

1.3 이항분포 (Binomial Distribution)

성공 확률이 $p$로 동일한 베르누이 시행을 $n$번 독립적으로 반복할 때, 성공 횟수 $X$의 확률분포입니다.

확률질량함수 (PMF)

$X \sim B(n,p)$
$p(x) = \binom{n}{x}p^x(1-p)^{n-x}, \quad x=0, 1, \dots, n$

기댓값 및 분산

  • 기댓값: $E(X) = np$
  • 분산: $Var(X) = np(1-p)$

p값에 따른 이항분포 그래프 ($n=7$)

모집단 크기(N)가 표본 크기(n)에 비해 매우 클 때, 초기하분포는 이항분포로 근사할 수 있습니다.

2. 정규분포 (Normal Distribution)

통계학에서 가장 중요한 연속확률분포로, 많은 실제 현상에서 나타나며 다른 분포의 근사로도 사용됩니다.

확률밀도함수 (PDF)

$p(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2\sigma^2}(x-\mu)^2}$
  • 평균 $\mu$를 중심으로 대칭인 종 모양을 가집니다.
  • 데이터의 약 68%는 $\mu \pm \sigma$ 범위에, 95%는 $\mu \pm 2\sigma$ 범위에, 99.7%는 $\mu \pm 3\sigma$ 범위에 존재합니다.

예제: 우유 용량

한 공장에서 생산하는 우유의 양이 평균 200mL, 표준편차 5mL인 정규분포 $N(200, 5^2)$를 따른다고 할 때, 우유 한 통의 양이 190mL보다 적을 확률을 구합니다.

1. 표준화: $Z = \frac{X-\mu}{\sigma} = \frac{190-200}{5} = -2$

2. 확률 계산: $P(X \le 190) = P(Z \le -2) \approx 0.0228$. 즉, 약 2.28%의 확률입니다.

3. 표본분포 (Sampling Distribution)

모집단에서 추출한 표본의 통계량(예: 표본평균)이 따르는 확률분포를 의미합니다.

중심극한정리 (Central Limit Theorem)

모집단의 분포와 상관없이, 표본 크기 $n$이 충분히 크면 (일반적으로 n≥30) 표본평균 $\bar{X}$의 분포는 근사적으로 정규분포를 따릅니다.

  • 표본평균의 평균: $E(\bar{X}) = \mu$
  • 표본평균의 분산: $Var(\bar{X}) = \sigma^2/n$
  • 따라서, $\bar{X} \approx N(\mu, \sigma^2/n)$

주요 표본분포: t, $\chi^2$, F 분포

t-분포 (t-Distribution)

모분산 $\sigma^2$을 모를 때 표본분산 $S^2$을 사용하여 모평균 $\mu$를 추론하는 데 사용됩니다. 정규분포와 유사하지만 꼬리가 더 두껍습니다. 표본 크기가 작을 때 유용합니다.

$T = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$

카이제곱분포 ($\chi^2$-Distribution)

표준정규분포를 따르는 확률 변수들의 제곱의 합이 따르는 분포로, 주로 모분산 $\sigma^2$을 추론하는 데 사용됩니다.

$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$

F-분포 (F-Distribution)

두 개의 독립적인 카이제곱 분포의 비율이 따르는 분포로, 두 정규모집단의 분산이 같은지 비교(분산비 검정)하는 데 사용됩니다.

$F = \frac{S_1^2 / \sigma_1^2}{S_2^2 / \sigma_2^2} \sim F(n_1-1, n_2-1)$

4. 정규분포 분위수대조도 (Q-Q Plot)

"지정된 확률분포(예: 정규분포)의 분위수"와 "자료분포의 분위수"를 산점도로 나타내어, 수집된 자료가 특정 분포를 따르는지 시각적으로 검토하는 방법입니다.

  • 해석: 산점도의 점들이 직선에 가깝게 분포하면 자료가 정규분포를 따른다고 볼 수 있습니다.
  • 점들이 S자 곡선을 그리거나 직선에서 체계적으로 벗어나면 정규성 가정을 만족하지 않는다고 판단할 수 있습니다.
