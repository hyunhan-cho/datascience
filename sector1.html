Chapter 01

STATA 시작하기

1. Stata 소개

1980년대 중반 StataCorp이 개발한 통계 S/W 패키지로, Statistics와 Data의 합성어입니다.

Stata의 장점

  • 데이터 관리와 그래픽에서 탁월한 능력 발휘
  • 광범위한 통계분석 가능
  • 인터넷을 통한 다양한 부가기능 이용
  • 타 통계프로그램에 비해 저렴한 비용

2. Stata 켜기: 창 (Window)

Command 창

명령어 입력

Result 창

명령 실행 결과 표시

Review 창

입력한 명령어 기록 (에러는 빨간색)

Variables 창

데이터 변수 목록 표시

3. 데이터 파일 불러오기 (1)

기존 데이터 파일을 불러오는 다양한 방법입니다.

  • 메뉴 이용: File > Open

  • 시스템 데이터 사용:

    sysuse auto, clear

  • 웹 데이터 사용:

    use http://www.stata-press.com/data/r9/auto.dta
webuse auto

3. 데이터 파일 불러오기 (2)

데이터를 직접 입력하거나 메모리에서 삭제합니다.

데이터 직접 입력

input 명령어를 이용합니다.

input aa bb cc
1 2 3
4 5 6
end

데이터 삭제

로딩된 모든 데이터를 삭제합니다.

clear

4 & 5. 디렉터리, 명령어와 대화창

STATA 관련 디렉터리 확인

sysdir

대화창(Dialog Box) 호출

db를 명령어 앞에 붙여 사용합니다.

db summarize

Option Syntax

명령어 뒤에 ,를 붙여 옵션을 추가합니다.

summarize price mpg, detail

6 & 7. 도움말과 작업환경

Help 메뉴 이용

help summarize

기본 설정 확인

set

메모리 설정

set memory 300m, perm

Output 설정

set more off, perm

8. 업데이트 및 명령어 요약

Stata 업데이트

모든 패키지를 최신 버전으로 업데이트합니다.

update all

파일 관리

  • sysuse, use, save, clear

데이터 확인

  • browse, edit, summarize

환경 설정

  • sysdir, set, help, update

기타

  • db, input
