등장 배경

AlexNet은 2012년 이미지넷(ImageNet) 대회에서 혜성처럼 등장했습니다. 당시 컴퓨터 비전 분야는 SIFT, HOG 같은 사람이 직접 설계한 특징 추출 알고리즘이 주류였습니다.

하지만 Alex Krizhevsky와 Geoffrey Hinton 등은 "데이터로부터 특징을 스스로 학습하는" 딥러닝 방식이 훨씬 강력함을 증명했습니다. 2위 모델(26.2%)과 비교해 무려 10% 이상의 격차(15.4%)를 벌리며 압도적인 우승을 차지했죠. 이는 딥러닝 붐을 일으키는 결정적인 사건이 되었습니다.

전체 구조 (Architecture)

[그림 1] AlexNet 구조 다이어그램 (입력 → 5개 합성곱층 → 3개 완전연결층)

  • Deep Architecture

    총 8개의 학습 가능한 계층으로 구성됨 (5개의 합성곱 층 + 3개의 완전 연결 층)

  • Two-Stream GPU Design

    당시 GPU 메모리(3GB) 한계를 극복하기 위해, 모델을 위/아래 두 경로로 나누어 병렬 학습시킴.

  • Input Size

    227 x 227 x 3 (RGB) 이미지를 입력으로 받음 (논문엔 224로 오기됨)

핵심 작동 원리 (Mechanism)

AlexNet이 높은 성능을 낼 수 있었던 3가지 주요 기술적 요인을 살펴봅니다.

Activation Function

1. ReLU (Rectified Linear Unit)

원리: 음수 입력은 0으로, 양수는 그대로 통과시킵니다.
f(x) = max(0, x)

효과: 기존 Sigmoid/Tanh 함수에서 발생하던 '기울기 소실(Gradient Vanishing)' 문제를 해결하고, 학습 수렴 속도를 6배나 가속화했습니다.

Regularization

2. Dropout (드롭아웃)

(클릭하여 시뮬레이션)

원리: 학습 시, 은닉층의 뉴런을 무작위 확률(50%)로 끕니다.

효과: 특정 뉴런에 의존하는 것을 막아 과적합(Overfitting)을 강력하게 방지합니다. 모델의 범용성을 높여줍니다.

Preprocessing

3. 데이터 증강 (Augmentation)

Original
Flipped
Bright
Cropped

원리: 이미지를 반전, 자르기(Crop), 색상 변환(PCA)하여 인위적으로 데이터를 늘립니다.

효과: 데이터셋을 2048배 뻥튀기하는 효과를 내어, 적은 데이터로도 과적합 없이 깊은 모델을 학습시킬 수 있었습니다.
