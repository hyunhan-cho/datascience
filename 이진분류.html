(1) 라이브러리 로딩

분석에 필요한 주요 라이브러리를 불러옵니다.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

from sklearn.model_selection import train_test_split
from sklearn.metrics import *
from sklearn.preprocessing import MinMaxScaler

from keras.models import Sequential
from keras.layers import Dense, Input
from keras.backend import clear_session
from keras.optimizers import Adam

1. 환경 준비

학습곡선 함수 정의

모델의 학습 과정을 시각화할 dl_history_plot 함수를 정의합니다.

def dl_history_plot(history):
    plt.figure(figsize=(10,6))
    plt.plot(history['loss'], label='train_err', marker = '.')
    plt.plot(history['val_loss'], label='val_err', marker = '.')

    plt.ylabel('Loss')
    plt.xlabel('Epoch')
    plt.legend()
    plt.grid()
    plt.show()

1. 환경 준비

(2) 데이터 로딩

titanic.3.csv 파일을 로드하고, 이번 분석에 사용하지 않을 컬럼들을 제거합니다.

path = "https://raw.githubusercontent.com/DA4BAM/dataset/master/titanic.3.csv"
data = pd.read_csv(path)
data.drop(['Age_scale1', 'AgeGroup', 'SibSp','Parch' ], axis = 1, inplace = True)
data.head()

2. 데이터 준비 (모델 1)

(1) 데이터 준비: Feature Selection

Sex, Age, Fare 3개 변수만 사용하여 Survived를 예측합니다.

target = 'Survived'
features = ['Sex', 'Age', 'Fare']
x = data.loc[:, features]
y = data.loc[:, target]

(2) 가변수화 (Dummification)

범주형 변수인 Sex를 숫자형으로 변환합니다 (One-Hot Encoding).

x = pd.get_dummies(x, columns = ['Sex'], drop_first = True)
x.head()

2. 데이터 준비 (모델 1)

(3) 데이터 분할 (Split)

데이터를 훈련용(70%)과 검증용(30%)으로 분리합니다.

x_train, x_val, y_train, y_val = train_test_split(x, y, test_size=.3, random_state = 20)

(4) 스케일링 (Scaling)

MinMaxScaler를 사용해 모든 특성을 0과 1 사이로 정규화합니다.

scaler = MinMaxScaler()
x_train = scaler.fit_transform(x_train)
x_val = scaler.transform(x_val)

3. 딥러닝 1: 3개 Feature

(1) 모델 설계

입력 (3개)에서 바로 출력 (1개)으로 이어지는 가장 단순한 모델입니다.

nfeatures = x_train.shape[1] # nfeatures = 3
clear_session()

model = Sequential( [Input(shape = (nfeatures,)),
                     Dense( 1, activation= 'sigmoid')])

model.summary()

3. 딥러닝 1: 3개 Feature

(2) Compile 및 학습

binary_crossentropy 손실 함수와 Adam 옵티마이저(학습률 0.01)로 50 Epochs 학습합니다.

model.compile(optimizer = Adam(learning_rate=0.01), loss = 'binary_crossentropy')

history = model.fit(x_train, y_train,
                    epochs = 50, 
                    validation_split=0.2).history
dl_history_plot(history)

3. 딥러닝 1: 3개 Feature

(3) 예측 및 검증

모델이 예측한 확률값을 0.5 기준으로 0 또는 1로 변환하여 실제값과 비교합니다.

pred = model.predict(x_val)
print(pred[:5])
# activation이 sigmoid --> 0 ~ 1 사이의 확률값.
# 그러므로 cut-off value(보통 0.5)를 기준으로 잘라서 0과 1로 만들어 준다.
pred = np.where(pred >= .5, 1, 0)
print(pred[:5])
print(classification_report(y_val, pred))

약 78%의 정확도(Accuracy)를 달성했습니다.

4. 딥러닝 2: 전체 Feature

(1) 데이터 준비

Survived를 제외한 모든 변수를 x로, Survived를 y로 사용합니다.

target = 'Survived'
x = data.drop(target, axis = 1)
y = data.loc[:, target]

(1) 가변수화

Pclass, Sex, Embarked 범주형 변수들을 get_dummies로 처리합니다.

cat_cols = ['Pclass','Sex', 'Embarked']
x = pd.get_dummies(x, columns = cat_cols, drop_first = True)

4. 딥러닝 2: 전체 Feature

(1) 데이터 전처리: 분할 및 스케일링

이전과 동일하게 70/30 (random_state=20)으로 분할하고 MinMaxScaler로 스케일링합니다.

x_train, x_val, y_train, y_val = train_test_split(x, y, test_size=.3, random_state = 20)
scaler = MinMaxScaler()
x_train = scaler.fit_transform(x_train)
x_val = scaler.transform(x_val)

4. 딥러닝 2: 전체 Feature

(2) 모델링

전체 Feature(n)를 사용하려 했으나, 모델 입력 shape이 nfeatures(3)로 설정되었습니다.

n = x_train.shape[1] # n은 현재 모든 Feature 수 (예: 10)
# nfeatures는 이전 3개 Feature 모델에서 넘어온 값 (현재 3)
clear_session()
model = Sequential( [Input(shape = (nfeatures,)), 
                     Dense( 1, activation = 'sigmoid')])
model.summary()

[주의] 모델의 입력 shape이 n이 아닌 nfeatures로 설정되어, 실제로는 3개의 Feature만 학습에 사용되었습니다. 이로 인해 모델 성능이 기대만큼 향상되지 않을 수 있습니다.

4. 딥러닝 2: 전체 Feature

Compile + 학습 및 예측

동일한 하이퍼파라미터로 학습 후, 예측 및 검증을 수행합니다.

model.compile(optimizer = Adam(learning_rate = 0.01), loss = 'binary_crossentropy')
history = model.fit(x_train, y_train, epochs = 50, validation_split=.2).history
dl_history_plot(history)

pred2 = model.predict(x_val)
pred2 = np.where( pred2 >= .5 , 1, 0)
print(classification_report(y_val, pred2))

입력 Feature가 제한되었기 때문에, 이전과 비슷한 약 78%의 정확도를 보였습니다.

5. 딥러닝 3: Hidden Layer 추가

(1) 모델 설계

모든 Feature(n)를 입력받고, 4개 노드의 relu 은닉층을 추가하여 모델의 복잡도를 높였습니다.

n = x_train.shape[1] # 모든 Feature 수
clear_session()

model3 = Sequential([
  Input(shape = (n,)),
  Dense(4, activation = 'relu'),
  Dense(1, activation = 'sigmoid')
])
model3.summary()

5. 딥러닝 3: Hidden Layer 추가

(2) Compile + 학습 및 검증

동일한 하이퍼파라미터로 학습 후, 예측 및 검증을 수행합니다.

model3.compile( optimizer=Adam(learning_rate= 0.01), loss ='binary_crossentropy')
hist = model3.fit(x_train, y_train, epochs = 50, validation_split=.2 ).history
dl_history_plot(hist)

pred3 = model3.predict(x_val)
pred3 = np.where(pred3 >= 0.5, 1, 0)
print(classification_report(y_val, pred3))

정확도 (Accuracy): ~81% (은닉층 추가와 모든 Feature 사용으로 성능 향상)

5. (5) 실습 3: 나만의 모델

"이번에는 여러분이 원하는 대로 모델을 설계하고, 학습해 봅시다!"

Tip: 은닉층(layer) 개수, 노드(node) 개수, 활성화 함수, 학습률(learning rate), Epochs 등을 다양하게 변경하며 최적의 모델을 찾아보세요.

예시 코드 구조:

# 1. 모델 설계
# n = x_train.shape[1]
clear_session()
my_model = Sequential([
    Input(shape = (n,)),
    Dense(..., activation = '...'),  # 원하는 은닉층 추가
    Dense(..., activation = '...'),  # 원하는 은닉층 추가
    Dense(1, activation = 'sigmoid')
])
my_model.summary()

# 2. 컴파일 및 학습
my_model.compile(optimizer = Adam(learning_rate = 0.001), loss = 'binary_crossentropy') # 학습률 조정 가능
my_history = my_model.fit(x_train, y_train, epochs = 100, validation_split = 0.2).history # Epochs 조정 가능
dl_history_plot(my_history)

# 3. 예측 및 검증
my_pred = my_model.predict(x_val)
my_pred = np.where(my_pred >= 0.5, 1, 0)
print(classification_report(y_val, my_pred))
