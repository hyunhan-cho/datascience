Chapter 06

데이터관리 고급명령어

1. tabulate, xi

더미변수(dummy variables)는 명목형 자료(범주형 변수)를 0과 1로 표현하는 방식입니다.

예) 인종 변수 → race

  • 백인인 경우 1, 아닌 경우 0
  • 흑인인 경우 1, 아닌 경우 0

tabulate와 xi 명령어를 이용해 쉽게 생성할 수 있습니다.

더미변수 만들기 I: tabulate

tabulate는 빈도표를 만드는 명령어지만, gen() 옵션을 통해 더미변수를 생성할 수 있습니다.

빈도표 확인

sysuse nlsw88, clear
tab race

더미변수 생성

tab race, gen(dum_race_)

dum_race_1, dum_race_2, dum_race_3 변수가 생성됩니다.

더미변수 만들기 II: xi

xi 명령어는 더미변수 생성을 위해 사용됩니다.

sysuse nlsw88, clear
xi i.race, noomit prefix(dum_)
  • i.race: race 변수를 기준으로 더미변수를 생성하라는 의미입니다. (i=indicator)
  • noomit: 모든 범주에 대한 더미변수를 생성합니다. (생략 시 기준범주 제외)
  • prefix(dum_): 생성될 변수 이름의 접두사를 지정합니다.

2. egen (Extended Generate)

egen은 gen의 확장형 명령어로, 그룹별 통계량 등 더 고차원적인 방법으로 새로운 변수를 생성할 때 사용합니다.

gen

수학적 변형, 변수간 결합

egen

그룹별 최대/최소, 평균, 비중 등

주요 egen 함수 (1)

그룹별 최대값, 최소값, 평균, 중앙값을 구하는 예제입니다.

회사별 최대 투자액 (Maximum)

by company, sort: egen max_invest = max(invest)

회사별 평균 투자액 (Mean)

by company, sort: egen mean_invest = mean(invest)

주요 egen 함수 (2)

그룹 내에서 각 관측치가 차지하는 비중(%)을 구합니다.

연도별 각 회사의 투자 비중 (Percent)

각 연도의 총 투자액 중 특정 회사의 투자액이 차지하는 비율을 계산합니다.

by year, sort: egen percent_invest = pc(invest)

5. 데이터 구조 예시: Wide vs Long

Wide Type

각 관측치가 하나의 개체, 반복 측정값이 여러 변수(column)로 표현됩니다.

idscore2023score2024
18590
27075

Long Type

각 관측치가 특정 시점의 측정값, 여러 행에 걸쳐 하나의 개체가 표현됩니다.

idyearscore
1202385
1202490
2202370
2202475

6. 파일 결합 예시: Append, Merge

Append: 세로 결합 (관측치 추가)

두 데이터 파일(`data1`, `data2`)을 세로로 합칩니다.

data1.dta

idscore
185
270

data2.dta

idscore
392
 
use data1.dta, clear
append using data2.dta

▼ 결과 ▼

idscore
185
270
392

Merge: 가로 결합 (변수 추가)

기준 변수(`id`)로 두 파일을 가로로 합칩니다.

master.dta

idname
1Kim
2Lee

scores.dta

idscore
185
270
 
use master.dta, clear
merge 1:1 id using scores.dta

▼ 결과 ▼

idnamescore_merge
1Kim853
2Lee703
