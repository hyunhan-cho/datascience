PyTorch Deep Dive

Deep Learning Framework for Dynamic Neural Networks

PyTorch란?

진화의 역사: 페이스북(Meta)이 Lua 언어로 된 Torch를 파이썬 버전으로 재개발하여 2017년에 공개했습니다.

핵심 철학: "파이썬답게(Pythonic)" 만들어져 직관적이고 유연합니다.

  • 초기에는 NumPy 같은 과학 연산 라이브러리로 시작
  • GPU를 이용한 텐서 조작 지원
  • 동적 신경망 구축 가능
  • 강력한 가속화된 계산 속도 제공
philosophy.py
# PyTorch Philosophy: Pythonic & Dynamic import torch # Define a tensor on GPU x = torch.randn(5, 3).cuda() # Dynamic Computation Graph if x.sum() > 0: y = x * 2 else: y = x - 1 print("Dynamic & Fast!")

파이토치의 핵심 구성요소

torch

메인 네임스페이스로, 텐서 조작 및 다양한 수학 함수를 포함합니다.

torch.autograd

신경망 학습의 핵심인 자동 미분(Automatic Differentiation) 기능을 제공합니다.

torch.nn

신경망(Neural Network) 구축을 위한 데이터 구조와 레이어 라이브러리입니다.

torch.optim

SGD 등 파라미터 최적화 알고리즘을 제공합니다.

torch.utils

데이터 로딩 및 조작 등 편리한 유틸리티 기능을 제공합니다.

torch.onnx

서로 다른 프레임워크 간 모델 공유를 위한 ONNX 포맷을 지원합니다.

텐서(Tensors)란?

tensor_basic.py
import torch # 1. Initialization data = [[1, 2], [3, 4]] x_data = torch.tensor(data) # 2. Attributes print(f"Shape: {x_data.shape}") # Output: torch.Size([2, 2]) print(f"Datatype: {x_data.dtype}") # Output: torch.int64 print(f"Device: {x_data.device}") # Output: cpu (or cuda:0)

데이터의 기본 컨테이너

텐서는 파이토치에서 데이터 표현을 위한 기본 구조입니다. 수치형 데이터를 저장하는 다차원 배열 컨테이너입니다.

  • NumPy의 ndarray와 매우 유사한 구조
  • CPU뿐만 아니라 GPU를 사용한 연산 가속이 가능
  • 딥러닝 모델의 입력, 출력, 매개변수 등을 모두 텐서로 처리

텐서의 기본 연산 (Operations)

수학 연산

기본적인 사칙연산과 내적 연산을 직관적으로 지원합니다.

  • torch.add(): 덧셈
  • torch.sub(): 뺄셈
  • torch.mul(): 곱셈
  • torch.div(): 나눗셈
  • torch.mm(): 내적 (Matmul)
  • svd: 특이값 분해 (U, S, V)

In-place 연산

연산 뒤에 _가 붙으면 결과가 텐서 자체에 즉시 저장됩니다.

y = x + y
x.copy_(y)
x.t_() # Transpose

메모리를 절약할 수 있지만, Autograd 사용 시 주의가 필요합니다.

텐서 조작: View & Reshape

형태(Shape) 변경

view는 텐서의 원소 개수를 유지하면서 크기나 모양을 변경합니다.

-1로 설정하면 남은 차원의 크기에 맞춰 자동으로 값을 계산하여 할당합니다.

예: 전체 20개 요소 중 한 차원이 5라면 나머지는 자동으로 4가 됩니다.

reshape.py
# 4x5 Matrix (Total 20 elements) x = torch.randn(4, 5) # Flatten to 1D vector y = x.view(20) # y.shape -> [20] # Infer dimension with -1 z = x.view(-1, 5) # 20 / 5 = 4 rows # z.shape -> [4, 5] # Error if dimensions don't match # err = x.view(3, 5) # 15 != 20

텐서 조작: 차원 변경

Squeeze (축소)

차원의 크기가 1인 차원을 제거(압축)하여 텐서의 차원을 줄입니다.

# [1, 2, 2] -> [2, 2]
x = torch.zeros(1, 2, 2)
y = x.squeeze()

Unsqueeze (확장)

특정 위치(dim)에 크기가 1인 차원을 생성하여 차원을 늘립니다.

# [2, 2] -> [1, 2, 2]
x = torch.zeros(2, 2)
y = x.unsqueeze(dim=0)

텐서 조작: 결합 (Combine)

Stack (쌓기)

새로운 차원을 생성하여 텐서들을 결합합니다. 지정된 차원에 따라 텐서들이 겹겹이 쌓이는 형태입니다.

Cat (Concatenate, 이어붙이기)

기존 차원을 기준으로 텐서들을 연결합니다. 차원의 수가 늘어나지 않고 해당 차원의 길이가 길어집니다.

combine.py
t1 = torch.tensor([1, 2]) t2 = torch.tensor([3, 4]) # Stack: Adds New Dimension s = torch.stack([t1, t2]) # [[1, 2], [3, 4]] -> Shape [2, 2] # Cat: Extends Existing Dimension c = torch.cat([t1, t2], dim=0) # [1, 2, 3, 4] -> Shape [4]

텐서 조작: 분할 (Split)

Chunk

텐서를 몇 개의 덩어리로 나눌 것인지 지정합니다.

# 텐서를 3개로 등분
torch.chunk(tensor, 3, dim=0)

Split

텐서를 특정 크기만큼 잘라냅니다.

# 크기가 3인 텐서들로 분할
torch.split(tensor, 3, dim=0)

PyTorch ↔ NumPy 호환성

numpy_compat.py
import numpy as np # 1. Zero Copy Mechanism a = torch.ones(5) b = a.numpy() # Modifying 'a' affects 'b' a.add_(1) print(b) # [2. 2. 2. 2. 2.] # 2. From Numpy n = np.array([1, 2]) t = torch.from_numpy(n)

메모리 공유 (Zero Copy)

CPU 상의 텐서와 넘파이 배열은 메모리 공간을 공유합니다. 하나를 변경하면 다른 하나도 즉시 변경됩니다.

  • .numpy(): 텐서를 넘파이 배열로 변환
  • torch.from_numpy(): 넘파이 배열을 텐서로 변환

* 주의: GPU 텐서(.cuda())는 메모리 영역이 달라 복사(Copy)가 발생합니다.
