Chapter 04

데이터파일 형식 전환

파일 형식 전환이란?

Stata의 기본 데이터 파일은 *.dta 형식입니다.

Excel, SPSS, SAS 등 다른 프로그램에서 생성된 파일을 Stata에서 사용하려면, Stata가 인식할 수 있는 *.dta 형식으로 변환하는 과정이 필요합니다.

형식 전환 방법

1. Stata Import 기능

Stata에 내장된 기능으로 ASCII, Excel, SAS, SPSS 등의 파일을 불러올 수 있습니다.

(이 프레젠테이션에서는 이 방법을 중심으로 설명합니다.)

2. Stat-Transfer 이용

다양한 데이터 형식 간의 상호 변환을 지원하는 별도의 전문 소프트웨어입니다.

5. 데이터 내보내기 (export)

반대로, Stata의 *.dta 파일을 다른 형식으로 저장할 수도 있습니다.

Export 기능 사용법

메뉴에서 File > Export를 선택하여 원하는 파일 형식을 지정하면 됩니다. (예: .txt, .csv, .xlsx 등)

데이터 호환성

Stata는 SAS, SPSS 등 다른 주요 통계 프로그램의 데이터 파일을 불러들이는 기능을 지원하여 데이터 호환성이 좋습니다.
