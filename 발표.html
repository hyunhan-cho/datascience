Project Overview
01. INTRODUCTION

기획 의도

계획은 세우지만 실천이 어려운 사용자들을 위해,
"건물을 한 층씩 쌓아 올리는(Floor)" 시각적 성취감을 제공합니다.


Target Audience

  • 딱딱한 To-do 리스트에 지친 사용자
  • 막연한 목표("살 빼기", "공부하기")만 있고 실행 계획이 없는 사람
  • RPG 게임처럼 성장하는 재미를 느끼고 싶은 사람

Core Concept

Schedule(건물) Floor(층) Completion(완공)

// UserProfileService.java
public void ensureSignupBonusOnFirstLogin(User user) {
    userProfileRepository.findById(user.getUserId())
        .orElseGet(() -> {
            UserProfile profile = new UserProfile();
            profile.setUser(user);
            
            // 가입 축하금 50코인 지급
            profile.setPoints(50); 
            profile.setPersonalLevel(1);
            
            return userProfileRepository.save(profile);
        });
}

Swagger UI (OpenAPI 3.0)

프론트엔드 협업을 위해 API 문서를 자동화했습니다.

  • /api/auth/** : 인증 (가입/로그인)
  • /api/schedules/** : 일정 관리
  • /api/floors/** : 투두/완료 처리
  • /api/me/** : 내 정보/프로필
// SecurityConfig.java
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) {
    http
        .csrf(csrf -> csrf.disable())
        .sessionManagement(sm -> sm.sessionCreationPolicy(STATELESS))
        .authorizeHttpRequests(auth -> auth
            .requestMatchers("/api/auth/**", "/swagger-ui/**").permitAll()
            .anyRequest().authenticated() // 그 외 모든 요청 인증 필요
        )
        .addFilterBefore(jwtAuthFilter, UsernamePasswordAuthFilter.class);
    return http.build();
}
Troubleshooting
08. ISSUES & SOLUTIONS

Issue 1: Character JSON Data Null

문제: 기존 가입 유저의 equipped_items 컬럼이 NULL이라 조회 시 500 에러 발생.

해결: @Column(nullable=true) 설정 및 마이그레이션 SQL 실행.

UPDATE characters SET equipped_items = '{}' WHERE equipped_items IS NULL;

Issue 2: AI Response Instability

문제: GPT 응답이 가끔 JSON 형식이 깨지거나 Markdown backtick(```)을 포함함.

해결: 문자열 전처리 로직 추가 및 예외 발생 시 로컬 알고리즘으로 대체하는 fallbackPlan 구현.

// JSON Cleaning Logic
if (cleaned.startsWith("```")) {
    cleaned = cleaned.substring(idx + 1, last);
}
