배치 정규화
(Batch Normalization)

딥러닝 모델 학습의 안전장치이자 가속 페달.
기울기 소실 문제를 해결하고 학습 속도를 획기적으로 높이는 비결을 알아봅니다.

배치 정규화란 무엇인가요?

배치 정규화(Batch Normalization)는 신경망의 각 층(Layer)에 들어가는 입력 데이터의 분포를 정규화(Normalization)하는 기법입니다. 쉽게 말해, 데이터가 신경망의 층을 통과할 때마다 제멋대로 퍼지는 것을 막고, "평균 0, 분산 1"의 예쁜 형태로 다듬어주는 과정입니다.

왜 필요한가요?

딥러닝 학습 시 기울기 폭주(Exploding Gradient)나 기울기 소실(Vanishing Gradient) 문제가 발생하면 학습이 아예 멈추거나 불안정해집니다. 배치 정규화는 이를 막아주는 강력한 방패 역할을 합니다.

내부 공변량 변화 (Internal Covariate Shift)

신경망은 여러 층으로 이루어져 있습니다. 학습이 진행되면서 앞쪽 층의 파라미터가 바뀌면, 뒤쪽 층으로 넘어오는 데이터의 분포도 계속 바뀝니다. 이를 내부 공변량 변화라고 합니다.

뒤쪽 층 입장에서는 "계속 입력값의 분포가 바뀌니까 적응하기 힘들어!"라고 불평하게 되죠. 배치 정규화는 이 분포를 일정하게 잡아주어 각 층이 안정적으로 학습할 수 있게 돕습니다.

수식으로 이해하기

Step 1. 평균 계산
\[\mu_{\mathcal{B}} \leftarrow \frac{1}{m} \sum_{i=1}^{m} x_{i}\]

배치 내 데이터의 평균을 구합니다.

Step 2. 분산 계산
\[\sigma_{\mathcal{B}}^{2} \leftarrow \frac{1}{m} \sum_{i=1}^{m} (x_{i} - \mu_{\mathcal{B}})^{2}\]

데이터가 평균에서 얼마나 퍼져있는지(분산) 구합니다.

핵심
Step 3. 정규화 (Normalize)
\[\hat{x}_{i} \leftarrow \frac{x_{i} - \mu_{\mathcal{B}}}{\sqrt{\sigma_{\mathcal{B}}^{2} + \epsilon}}\]

데이터를 평균 0, 분산 1로 만듭니다.
(\(\epsilon\)은 0으로 나누는 것을 방지하는 아주 작은 수)

Step 4. 스케일 & 시프트
\[y_{i} \leftarrow \gamma \hat{x}_{i} + \beta\]

데이터의 고유한 특성을 잃지 않도록
학습 가능한 파라미터 \(\gamma\)(스케일), \(\beta\)(시프트)를 추가합니다.

초기화 팁: 보통 학습 시작 시 \(\gamma\)는 1, \(\beta\)는 0으로 초기화됩니다. 이 값들은 역전파(Backpropagation) 과정에서 학습을 통해 최적의 값으로 갱신됩니다.

실제 계산 예시

아래와 같은 입력 데이터 텐서(Tensor)가 있다고 가정해 봅시다. 각 열(Column)을 하나의 특성(Feature)으로 보고 배치 정규화를 수행합니다.

import torch
x = torch.FloatTensor([
[-0.6577, -0.5797, 0.6360],
[ 0.7392, 0.2145, 1.5230],
[ 0.2432, 0.5662, 0.3220]
])

1 첫 번째 열(Column 0) 계산 과정

입력값 (\(x\))
-0.6577 0.7392 0.2432
평균(\(\mu\)): 위 세 값의 평균은 약 0.1082 입니다.
분산(\(\sigma^2\)): 평균과의 차이를 제곱하여 평균을 내면 약 0.3340 입니다.
결과값 (\(\hat{x}\))
-1.3246 1.0912 0.2334

* 정규화 후, 이 데이터들의 평균은 0, 분산은 1이 됩니다.

PyTorch 실행 결과

tensor([[-1.3246, -1.3492, -0.3756],
        [ 1.0912,  0.3077,  1.3685],
        [ 0.2334,  1.0415, -0.9930]])
